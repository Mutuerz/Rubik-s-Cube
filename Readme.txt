Steps
1. Execute the play_me script.
2. A resolved rubik´s cube will appear, you have 5 seconds to orientate the cube into a position in which you are able to see three
of the faces. Is recommended to focus camera on a vertice.
3. After 5 seconds delay the cube will shuffle randomly.
4. Immediatly after, it will start creating trees and looking for best moves.
5. This process will take a while.
6. When the cube is resolve you make look in the directory in where you ran the script, there you will find a Results.txt, this file
will contain the steps taken to achive completion, also, the heuristic value found in that position.
